.. _power:

Power pipeline
**************

The power pipeline computes the power spectral density (PSD) on epochs or raw data
on sensor space. 

The input data has to be in **fif** format. 

The output are the **psd tensor and frequencies in .npz format** stored in the workflow directory. 

The power pipeline is implemented by the function :py:func:`create_pipeline_power <neuropype_ephy.pipelines.power.create_pipeline_power>`
and its Node (:py:class:`power <neuropype_ephy.interfaces.mne.power.Power>`) 
wraps the MNE Python functions  |welch|, |multitaper| computing the PSD
using Welch's method and multitapers respectively.
